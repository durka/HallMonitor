HallMonitor in System App

Tested on CM and PA ROMS.

Feature:
- Able to use sleep mode instead of lock mode.

Before install HallMonitor to System App for the first time:
- Go to Settings/Apps: Clear Data, Clear Cache, uninstall update and uninstall completly HallMonitor
- Reboot to Recovery
- Install HallMonitor using zip file
- Reboot to CM/PA, wait full start
- Launch and configure HallMonitor
